1. О программе HyperTest

Программа предназначена для проверки знаний (тестирования).
В отличии от других программ, она обладает следующими 
характерными свойствами:
- возможно включение базы с 512 вопросами;
- количество ответов на вопрос - до 20;
- можно выбирать несколько ответов;
- можно возвращаться к предыдущим вопросам;
- гибкая система оценки;
- просмотр и печать протокола тестирования;
- и др.
Первые версии программы, как тестовые будут условно 
бесплатными, в будущем намечается переход на коммерческую 
основу. Автор программы - Рыбалкин К.Г. (Караганда, 
Казахстан). Все права на программу HyperTest принадлежат 
только автору.

2. Общая информация


2.1 Формат тестовой базы.

Тестовая база может находится в файлах .ASK (незакодированная
база) и .CSK (закодированная). Формат файла .ASK должен строго
соответствовать следующим правилам:
1. В начале файла, сразу после тэга <discipline>, идет 
   название дисциплины, по которому проводится тестирование.
2. Начало вопроса начинается с символов "/\", затем 
   непосредственно сразу после этих символов идет вопрос.
   С версии 1.0b вопросы и ответы можно вводить в несколько 
   строк, но при показе вопроса в окне тестирования перенос 
   учитываться не будет. Для принудительного переноса строки 
   необходимо использовать HTML тэг <br>.

   Пример:
   
/\Вопрос 1
- Ответ 1
+ Строка с принудительным<br>
  переносом
- Строка 
  с переносом
\/

При тестировании ответы будут показываться следующим образом:

Вопрос 1

[] Ответ 1
[] Строка с принудительным
   переносом
[] Строка с переносом


3. Далее следуют ответы, которые начинаются с символов "+" 
   или "-", соответственно правильный ответ или неправильный.
   Также как и вопрос, ответ может идти в несколько строк.
4. Начиная с версии 1.0a появилась еще одна опция базы, 
   "примечание". Примечание может как присутствовать, так и
   отсутствовать - это не обязательный параметр. Для вставки
   примечания в вопрос, сразу после ответов, на новой строке
   необходимы символы "@@", после которых и идет текст 
   примечачания, например: 
   @@ См. раздел VI, "Денежное образение и денежная система"
5. Вопрос, в совокупности с ответами и комментариями 
   заканчивается символами "\/"

Пример такой структуры показан в прилагаемом файле TEST.ASK,
который идет в поставке с программой. Файлы .CSK можно
сформировать, выбрав в меню "База вопросов" пункт "Сохранить
как .CSK (закодированная база)...", предварительно нажав
из окна настроек на кнопку "Просмотр и редактирование базы".
ВНИМАНИЕ! При потере пароля данные восстановить будет 
невозможно!

2.2 Как подсчитываются проценты

Если в вопросе содержаться только два ответа, то проценты 
насчитываются простым сложением с совокупным итоговым 
процентом +100% и -100%. Если же содержится несколько 
правильных или неправильных ответов, то подсчет ведется 
следующим образом:
- за первый (не)правильный ответ (отнимается)прибавляется 50%;
- остальные 50% распределяются по другим (не)правильным ответам 
  по-ровну;
После окончания теста, в соответстви с установленными
процентами, характеризующими оценку, программа выдает 
результат в виде оценок "неудовлетворительно", 
"удовлетворительно", "хорошо", "отлично".

2.3 Совместимость с Windows'95

Если программа по каким-либо причинам не запускается в среде
Windows'95, то единственным решением данной проблемы является
обновление от Microsoft - "Microsoft comctl32.dll Update".
Ссылка на файл с обновлением:
http://hypertest.virtualave.net/download/40comupd.zip

3. В будущих версиях

Возможна реализация следующих функций:
- одновременное использование нескольких баз с вопросами
- файл языковой поддержки (позволит изменять интерфейс с 
  русского языка на любой другой)
- вставка в ответы и вопросы, наряду с текстом, также и 
  рисунков (можно осуществлять и сейчас, используя HTML тэг
  <img>, но в дальнейшем вставка рисунков будет упрощена);
- редактор баз вопросов в качестве отдельной утилиты;
- реализация ваших пожеланий.
